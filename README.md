AI Budget Planner

Описание проекта  
AI Budget Planner — это веб-приложение для прогнозирования расходов компании по категориям с использованием Machine Learning и генеративного AI.

Пользователь загружает CSV-файл с историческими данными, после чего система:
- строит прогноз расходов на следующий период  
- визуализирует динамику  
- генерирует аналитический бизнес-отчёт с рекомендациями

Используемые технологии  

Machine Learning:
- RandomForestRegressor  
- Feature Engineering:
  - lag-фичи (lag_1, lag_2, lag_3)  
  - rolling mean  
  - сезонность (sin/cos)  
- ColumnTransformer:
  - OneHotEncoder  
  - StandardScaler  

AI:
- Google Gemini API  
- Генерация бизнес-анализа и рекомендаций  

Интерфейс:
- Streamlit  

---

Входные данные  

CSV-файл должен содержать:

date — дата  
category — категория расходов  
amount_kzt — сумма  

---

Как работает модель  

1. Данные агрегируются по категориям и датам  
2. Генерируются признаки (месяц, год, лаги, сезонность)  
3. Модель прогнозирует расходы на следующий месяц  
4. AI анализирует результат и формирует отчёт
